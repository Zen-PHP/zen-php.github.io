Vagrant+VirtualBox搭建LNMP开发环境

发表于 2016-08-03

标签（空格分隔）： 工具

经历过windows系统下WAMP等集成开发环境和VirtualBox虚拟机运行Linux系统开发之后，发现Vagrant+virtualbox搭建PHP运行环境是比较好的

一、资源下载

（一）下载VirtualBox

  • 官方下载：VirtualBox

    （二）下载Vagrant

  • 官方下载：Vagrant

    （三）下载系统镜像

  • 官方下载：VagrantBox
  • 推荐版本：Official Ubuntu 12.04 daily Cloud Image amd64 (VirtualBox 4.1.12)

二、安装配置

（一）安装

1、安装VirtualBox+Vagrant

PS:安装vagrant之后需要重启电脑完成环境变量设置等配置

2、新建工作目录，如：

`D:VagrantWorkcentos-6.6-x86_64`

3、打开CMD命令行，进入新建工作目录

` D:`
`cd D:VagrantWorkcentos-6.6-x86_64`

4、初始化命令：

`vagrant init centos6.6`

5、手动加载BOX

  • 把下载的centos-6.6-x86_64.box复制到本目录D:VagrantWorkcentos-6.6-x86_64下，执行:
    vagrant box add centos6.6 centos-6.6-x86_64.box

6、BOX列表（检查是否导入成功）：

`vagrant box list`

（二）配置

PS:安装系统成功之后，会自动生成一个vagrantfile文件，是系统配置文件

1、端口映射：

`config.vm.network :forwarded_port, guest: 80, host: 8080`
上面这句代码前面的#号去掉。它表示映射本机的8080端口到虚拟机的80端口

2、目录映射：

config.vm.synced_folder "D:/www", "/var/www/html"
表示把本机的data目录共享到虚拟机里的/var/www目录

三、基本操作

  1. 启动
    vagrant up
  2. 关机
    vagrant halt
  3. 暂停
    vagrant suspend
  4. 恢复
    vagrant resume
  5. 移除
    vagrant box remove centos6.6
  6. 删除
    vagrant destroy
  7. 导出
    vagrant package
  8. 连接
  • DOS下vagrant ssh直接连接
  • 第三方终端模拟器连接（windows下支持）
