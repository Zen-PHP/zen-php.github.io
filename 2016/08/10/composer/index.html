Composer的简单使用

发表于 2016-08-10

Composer是PHP的依赖关系管理工具，而不是严格的PHP包管理工具

安装php-cli

sudo apt-get install php5-cli

下载Composer

sudo curl -sS https://getcomposer.org/installer | sudo php

安装Composer

/usr/bin/php composer.phar --version

设置全局命令

sudo mv composer.phar /usr/local/bin/composer

查看是否安装与设置成功

sudo composer -version

没错，如果安装成功，会出现一个很装逼的大写的拼装的COMPOSER

安装过程中可能会出现的问题

  • 在安装过程中 Composer 可能会询问你 GitHub 账户的登录信息，因为可能在使用中超过了 GitHub API （对匿名用户的）使用限制
